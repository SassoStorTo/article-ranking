# brief 5
## Title
Article Selection and Ranking

## Given Description
Given a list of N articles about the same story, design and implement an algorithm
that selects the "best" M. The core of the project is formally defining what "best"
means — informative? diverse from the others? low on opinion? more recent or
local? The group must propose at least 2–3 different criteria, implement them, and
compare the resulting rankings. Input: N articles grouped by event. Output:
ranking according to multiple criteria + comparative analysis. Evaluation:
comparison across criteria and a minimal user study (asking 5–10 people which
selection they prefer).
